# monorepo-todo

> An example monorepo using Vite, React, tRPC and Typescript to create a multi-app monorepo

## setup

```bash
# make sure you have turbo
pnpm add -g turbo

# clone the repo
git clone git@github.com:rayzr522/monorepo-todo.git
cd monorepo-todo

# install dependencies
pnpm install

# run the dev server
turbo dev
```
